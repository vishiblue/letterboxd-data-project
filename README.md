# Letterboxd Analytics Project

This project explores movie ratings and reviews from the movie platform Letterboxd.
The goal is to build practical data science skills by analyzing patterns in movie ratings, reviews, and popularity.

The project simulates the workflow of a data scientist working on a movie platform.

It is part of the project to upskill myself after a PhD in pure mathematics. 

---

## Project Goals

 It aims to explore questions such as:

* How are movie ratings distributed on Letterboxd?
* Which films generate the most engagement?
* Do more popular films receive higher ratings?

The deliverable is going to include an analytical report and an interactive dashboard. 

Later stages of the project will also explore statistical modeling, causal inference analysis, machine learning models and AI-assisted analysis of movie reviews.

---

## Planned Workflow

The project follows a full data science workflow:

1. Data loading and cleaning
2. Exploratory Data Analysis (EDA)
3. Metrics design (popularity, engagement, rating stability)
4. Statistical modeling of movie ratings
5. Causal inference analysis
6. Machine learning models
7. AI-assisted analysis of movie reviews
